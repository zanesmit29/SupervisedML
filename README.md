# Supervised Machine Learning

An educational repository documenting my journey to refine my understanding of Supervised Machine Learning concepts and practical implementations.

## Overview

This repository contains Jupyter notebooks covering fundamental and advanced supervised learning techniques, developed as part of the Machine Learning Specialization by DeepLearning.AI.

## Topics Covered

### Supervised ML
- Linear regression
- Logistic regression  
- Gradient descent optimization
- Feature scaling and engineering

### Advanced Algorithms
- Neural networks architecture and training
- Forward and backward propagation
- Model evaluation and selection
- Decision trees
- Practical ML system design

## Libraries Used

- **NumPy**: Numerical computing and array operations
- **Matplotlib**: Data visualization
- **scikit-learn**: Classical ML algorithms and preprocessing
- **TensorFlow/Keras**: Neural network implementation and training

## Repository Structure

```
src/
├── Supervised ML/       # Core supervised learning concepts
├── Advanced Algorithms/ # Neural networks and advanced topics
└── Extra/              # Supplementary materials
```

## Purpose

This repository serves as a learning reference and demonstrates practical applications of supervised machine learning techniques, from basic regression to advanced neural networks.
